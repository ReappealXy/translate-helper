# 多语言翻译助手

## 📝 项目介绍

这是一个强大的网页翻译助手，支持作为浏览器扩展或独立JavaScript脚本运行。它允许你在任何网页上选择文本并进行即时翻译，支持多种语言互译，无需跳转到翻译网站。

**作者**: Rexy  
**官网**: [www.rexy.xin](https://www.rexy.xin)

## 🚀 安装方法

本翻译助手提供三种使用方式：

### 1. 作为Chrome/Edge浏览器扩展安装（推荐）

1. 下载并解压缩扩展文件
2. 打开浏览器，进入扩展管理页面：
   - Chrome: 访问 `chrome://extensions/`
   - Edge: 访问 `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展"按钮
5. 选择解压后的扩展文件夹
6. 扩展即安装完成，浏览器右上角会出现扩展图标

### 2. 直接在本地网页中测试

1. 下载项目文件（`content.js`和`index.html`）
2. 打开`index.html`文件
3. 测试翻译功能（选择文本，使用快捷键或点击悬浮图标）

### 3. 作为油猴(Tampermonkey)脚本使用

1. 安装[Tampermonkey](https://www.tampermonkey.net/)浏览器扩展
2. 创建新脚本，并将`content.js`中的代码复制粘贴进去
3. 保存脚本
4. 现在可以在任意网页上使用翻译功能了！

## 🔧 使用方法

1. **选择文本**：在任何网页上选择你想要翻译的文本
2. **呼出翻译面板**：
   - 使用快捷键：`Ctrl + Shift + T`
   - 或点击浏览器工具栏上的扩展图标（如果使用扩展方式）
   - 或点击网页右下角悬浮的📘按钮
3. **选择语言**：
   - 系统会自动检测输入文本的语言
   - 也可以手动选择源语言和目标语言
   - 使用⇄按钮可以快速交换源语言和目标语言
4. **翻译**：点击"翻译"按钮，获取翻译结果
5. **拖动**：通过面板顶部进行拖动
6. **关闭**：点击右上角×按钮关闭面板

## ✨ 功能特点

- 支持11种主流语言互译：
  - 中文、英语、日语、韩语、法语
  - 德语、西班牙语、俄语、葡萄牙语
  - 意大利语、阿拉伯语
- 自动语言检测功能
- 长文本分段翻译（突破API字符限制）
- 三种使用方式：浏览器扩展、网页脚本、油猴脚本
- 支持快捷键和悬浮按钮双触发方式
- 简洁美观的界面
- 面板可拖动
- 纯JavaScript实现，无需外部依赖

## ⚙️ 自定义配置

可以在`content.js`（或`translator.js`）文件开头的`config`对象中修改以下配置：

```javascript
const config = {
  shortcutKey: { ctrl: true, shift: true, key: 'T' }, // 快捷键设置
  floatButtonText: '📘',                              // 悬浮按钮图标
  apiUrl: 'https://api.mymemory.translated.net/get',  // 翻译API地址
  languages: {                                        // 支持的语言
    'zh': '中文',
    'en': '英语',
    'ja': '日语',
    // ... 更多语言
  },
  autoDetectThreshold: 0.7,                           // 语言检测阈值
  maxCharsPerRequest: 500                             // 每次翻译请求的最大字符数
};
```

## 🔍 技术说明

- 使用原生JavaScript DOM操作
- 采用MyMemory免费翻译API
- 基于字符集特征的语言自动检测
- 智能文本分段处理长文本翻译
- 所有样式通过JavaScript动态注入，无需额外CSS文件
- 面板拖动使用原生鼠标事件实现
- 支持Chrome扩展Manifest V3规范

## 📌 注意事项

- 此翻译助手使用的是免费API，可能有调用频率限制
- 某些网站可能因安全策略限制脚本运行
- 为获得最佳体验，建议在Chrome或Edge等现代浏览器中使用
- 自动语言检测基于字符特征，对混合语言文本可能检测不准确
- 扩展版本需要图标文件(16x16, 48x48, 128x128像素)放在images文件夹中

- **index.html**: 测试用HTML页面
- **translator.js**: 主要功能实现脚本
